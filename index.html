Breast Cancer Subtype Classification using Machine Learning

2025 • University of Windsor

Developed a machine learning framework for breast cancer subtype classification using the METABRIC clinical dataset. The project combined preprocessing, SMOTE balancing, feature selection, and model optimization using Support Vector Machines and Random Forest classifiers.

The optimized Random Forest model achieved 99.37% accuracy and 99.40% F1-score, significantly outperforming SVM-based approaches.

Feature Importance Analysis: Mutual information scoring identified highly predictive clinical biomarkers including Oncotree Code and Tumour Histologic Subtype.
